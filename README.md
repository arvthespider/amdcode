# Tiara — Graphic Design Portfolio

Portfolio website for **Azizah Mutiara Dewi**, built with Next.js 15. Single-page design with glassmorphism UI, dark/light mode, and Framer Motion animations.

---

## Tech Stack

| Layer | Library |
|---|---|
| Framework | Next.js 15 (App Router) |
| UI | React 19 + Tailwind CSS 3 |
| Animations | Framer Motion 11 |
| Icons | Lucide React |
| Fonts | Poppins (body) + Montserrat (display) via Google Fonts |
| Analytics | Vercel Analytics |
| Language | TypeScript |

## Styling

### Colors / CSS Variables

`app/globals.css` — top section with `:root` and `.dark` variables:
- `--background` — page background color
- `--line` — border/divider color
- `--glass-bg`, `--glass-border` — glassmorphism card appearance

### Fonts

`app/layout.tsx:6–16` — Poppins and Montserrat loaded via Next.js Google Fonts. Change weights or families here.

`tailwind.config.ts` — maps CSS variables to Tailwind classes:
- `font-sans` → Poppins
- `font-display` → Montserrat

### Dark Mode

Toggled by adding/removing `class="dark"` on `<html>`. Preference saved to `localStorage` key `"tiara-theme"`. Logic in `ThemeToggle` component, `portfolio-page.tsx:133`.

## Adding New Sections

1. Add new `<section>` block inside the `<main>` in `PortfolioPage` (`portfolio-page.tsx:379`)
2. Use existing components for consistency:
   - `<Reveal>` — scroll-triggered fade-in animation
   - `<SpotlightCard>` — card with mouse spotlight effect
   - `<SectionHeading kicker="" title="" description="">` — standard section header
3. Add nav link in navbar (`portfolio-page.tsx:403`) and footer (`portfolio-page.tsx:1058`)

## Contact Form Behavior

The form does **not** use a backend. On submit it opens the user's email client with fields pre-filled via `mailto:`. To add real form submission, replace `handleSubmit` in `ContactForm` (`portfolio-page.tsx:278`) with a fetch to a form service (Formspree, Resend, etc.).
